Cursor is an AI-powered code editor built on VS Code. It gives you an AI assistant that can read your project, write code, and run commands. Even if you're a designer, Cursor is a powerful tool for automating repetitive tasks, generating prototypes, and working with design tools like Figma.

## Installation

<Steps>

1. **Download Cursor**

   Go to [cursor.com](https://www.cursor.com/) and download the installer for your operating system.

2. **Install and launch**

   Run the installer and open Cursor. If you've used VS Code before, Cursor will feel immediately familiar since it's built on the same foundation.

3. **Sign in**

   Create a Cursor account or sign in with your existing account. The free tier gives you access to AI features with usage limits.

4. **Import your VS Code settings** (optional)

   If you already use VS Code, Cursor can import your extensions, themes, and keybindings automatically during first launch.

</Steps>

## Key Concepts

### The AI Chat Panel

Press `Cmd + L` (Mac) or `Ctrl + L` (Windows) to open the AI chat panel. This is where you talk to the AI assistant. You can ask it questions, give it instructions, or paste in context.

### Agent Mode

Agent mode lets the AI make changes to your files, run terminal commands, and work through multi-step tasks autonomously. Toggle it from the chat panel dropdown.

### MCP Servers

MCP (Model Context Protocol) servers extend what Cursor's AI can do. They let the AI connect to external tools like Figma, Slack, databases, and more. You'll set these up in the guides that follow.

## Recommended Settings

Open settings with `Cmd + ,` and consider these adjustments:

- **Auto-save:** Enable auto-save so your files are always current when the AI reads them
- **Terminal:** Make sure the integrated terminal is accessible (`Ctrl + ~`)
- **Theme:** Pick a theme you're comfortable with since you'll be spending time here
