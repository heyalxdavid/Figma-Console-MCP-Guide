## Why Cursor?

Cursor is an AI-powered code editor based on VS Code. It gives you a full IDE experience with an agent that can read, write, and modify your project files directly.

The biggest advantage for designers is **visibility**. You can see your entire folder structure, open any file, and watch changes happen in real time as the agent works. If it creates a new component or edits a config file, you'll see it immediately in the sidebar. This makes it much easier to understand what's going on in your project, especially when you're learning.

Cursor also supports Claude as its underlying AI model, so you get the same intelligence as the Claude CLI or Desktop app, but wrapped in a more visual, approachable environment.

<Aside type="tip">
If you're new to coding tools, Cursor is the recommended starting point. The visual file browser and editor make it the easiest way to follow along with what the AI is doing.
</Aside>

## Installation

Download and install Cursor from [cursor.com](https://cursor.com). Follow the on-screen setup to sign in and configure your editor preferences.
