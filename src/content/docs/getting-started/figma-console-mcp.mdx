The Figma Console MCP (by [Joey Banks / southleft](https://github.com/southleft/figma-console-mcp)) goes beyond reading designs. It lets Cursor's AI assistant execute Figma Plugin API code directly, so it can create frames, add text, modify styles, search components, and even work with FigJam boards.

## Prerequisites

- [Cursor installed](/getting-started/cursor-setup/)
- [Figma MCP configured](/getting-started/figma-mcp/) (recommended but not required)
- Figma Desktop app installed

## How It Works

Unlike the standard Figma MCP (which reads files via the Figma REST API), the Console MCP works through a **Desktop Bridge plugin** that runs inside Figma. The bridge creates a WebSocket connection between Figma and the MCP server, letting the AI execute code in the context of your open Figma file.

## Setup

<Steps>

1. **Install the MCP server**

   Add the Figma Console MCP to your Cursor MCP settings. Go to **Settings > MCP** and add:

   ```json
   {
     "mcpServers": {
       "figma-console": {
         "command": "npx",
         "args": [
           "-y",
           "@anthropic/figma-console-mcp"
         ]
       }
     }
   }
   ```

   :::note
   Check the [official repo](https://github.com/southleft/figma-console-mcp) for the latest package name and installation instructions, as these may change.
   :::

2. **Install the Desktop Bridge plugin in Figma**

   Open Figma Desktop, go to **Plugins > Development > Import plugin from manifest**, and point it to the bridge plugin included with the MCP server. Alternatively, search the Figma Community for "Desktop Bridge."

3. **Run the bridge plugin**

   Open the Figma file you want to work with, then run the Desktop Bridge plugin from **Plugins > Development > Figma Desktop Bridge**. The plugin will connect to the MCP server.

4. **Verify in Cursor**

   Back in Cursor, check your MCP settings. The Figma Console MCP should show as connected. You can test it by asking the AI to describe what's on the current Figma page.

</Steps>

## What You Can Do

With the Figma Console MCP connected, you can ask Cursor to:

- **Create frames and shapes:** "Create a 1440x900 frame with a 16px grid"
- **Add and style text:** "Add a heading that says 'Dashboard' using Inter Bold 32px"
- **Search components:** "Find all button components in this file"
- **Modify properties:** "Change the fill color of the selected element to #3B82F6"
- **Take screenshots:** "Take a screenshot of the current page"
- **Work with FigJam:** Create sticky notes, flowcharts, tables, and more on FigJam boards

## Troubleshooting

- **Bridge not connecting:** Make sure you're using Figma Desktop (not the browser version). The bridge plugin requires the desktop app.
- **Plugin not found:** If you can't find the bridge plugin, try reinstalling from the official repo or Figma Community.
- **Commands failing:** The AI executes Figma Plugin API code. Some operations require specific node selections or page context. Try selecting the target element first.
