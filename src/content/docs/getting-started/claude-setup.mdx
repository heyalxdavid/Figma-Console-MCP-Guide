## Why Claude?

Claude is available as a **CLI** (command-line interface) and a **Desktop app**. Both let you chat with an AI agent that can create and edit files on your computer.

- **Claude CLI** runs in your terminal. It's fast and lightweight, great for quick tasks or if you're comfortable working in a terminal.
- **Claude Desktop** is a standalone app with a conversational UI. It feels more like chatting with an assistant than using a code editor.

Both tools are simple to get started with, but they trade off some visibility compared to Cursor. With the CLI, you're working in a terminal window, so you won't see a file browser or live preview of changes. With the Desktop app, you can see the conversation and generated code, but you won't have a full view of your project's folder structure unless you also have Finder open alongside it.

<Aside type="note">
If you prefer a more visual, IDE-like experience where you can browse files and see every change as it happens, check out [Getting Started with Cursor](../cursor-setup/) instead. You can use Claude as the AI model inside Cursor, giving you the best of both worlds.
</Aside>

## Installation

Install Claude from [claude.ai/download](https://claude.ai/download) for the Desktop app, or install Claude Code via npm for the CLI:

```bash
npm install -g @anthropic-ai/claude-code
```

Follow the prompts to sign in with your Anthropic account.
