title: Installing Homebrew

---

Homebrew is a package manager for macOS that makes it easy to install developer tools like Node.js, Git, and more. If you already have it installed, you can skip this page.

## Check if you already have it

Open **Terminal** (search for "Terminal" in Spotlight with <kbd>Cmd</kbd> + <kbd>Space</kbd>) and run:

```
brew --version
```

If you see a version number, you're all set. If you see "command not found", follow the steps below.

## Install Homebrew

Go to [brew.sh](https://brew.sh) and copy the install command from the homepage. Paste it into your terminal and press Enter.

Follow the prompts to complete the installation. When it finishes, it may ask you to run a couple of commands to add Homebrew to your PATH. Copy and run those commands.

Verify it's installed:

```
brew --version
```
