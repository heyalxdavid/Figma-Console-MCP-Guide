# Figma Console MCP Guide

A getting-started guide for designers working with AI tools, Figma MCPs, and code assistants like Cursor and Claude. Covers setting up the Figma MCP (read access) and Figma Console MCP (read + write access), along with prerequisites like Homebrew and Node.js.
